## 《星尘的研究》COC TRPG replay

2012年12月，一封来自英国伦敦贝克街221B博物馆管理员的求助信打破了圣诞前夕的平静。那封邮件寻找能帮助寻回被偷走馆藏的人，并且发信人宣称，那件本属于伟大侦探的物品很可能会导致世界末日的到来，而她仅有的线索指向了美国马萨诸塞州的阿卡姆市。
